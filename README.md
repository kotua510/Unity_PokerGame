## 1.説明
初期の持ち金を1000としてポーカーを遊べるゲームをUnity及びC#を用いて作成しました。<br>
本ゲームは文化祭での公開を前提としており、ポーカーのルールを知らない人など、誰でも手軽にプレイできる点を重視して開発しました。<br>
文化祭でカジノゲームを制作するプロジェクトに参加していたため、その一つとして本ゲームを作成しました。<br>

## 2.使用技術
C#,Unity

## 3.製作期間
2025年 9月~11月

## 4.アピールポイント、工夫点など
・アピールポイント①:手軽にプレイできる設計<br>
文化祭での展示を想定し、短時間かつ直感的に遊べるよう設計しました。具体的には、ポーカーのルールを一部簡略化し、掛け金の設定を入力ではなくボタン操作にすることで、ポーカーが初めてのプレイヤーでも迷わずプレイできるようにしました。<br>
・アピールポイント②:他プレイヤーの情報漏洩を防ぐUI設計<br>
本ゲームは1台の端末で複数人がプレイする仕様であるため、プレイヤー切り替え時に手札が見えてしまう課題がありました。これを防ぐため、プレイヤー切り替え時に確認画面を表示する仕組みを実装し、公平性を保ちました。<br>

## 5.プレイしていただく時の注意点
カジノプロジェクトの一部をそのまま1つのゲームしているのでゲームの最後の「やめる」ボタンは本来タイトルに戻るのですが、ここにあるexeファイルでは「続ける」ボタンと同じく、人数選択画面に戻るだけですのでご了承ください。<br>
